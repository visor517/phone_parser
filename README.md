# Парсер телефонных номеров

Инструмент для парсинга диапазонов телефонных номеров с сайта [kody.su](https://www.kody.su/mobile), операторов и регионов, с сохранением данных в JSON.

## 📦 Функционал

- Парсит коды от `900` до `999` (формат `+7 9XX XXX-XX-XX`)
- Извлекает:
  - Диапазоны номеров (`from`, `to`)
  - Оператора связи
  - Регион или город
  - Временную зону (на основе `timezones.py`)
- Сохраняет результат в `results/DEF-9xx.json`

## 🚀 Запуск

1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите скрипт: `python run.py`
3. Результаты в папке `results/DEF-9xx.json`
   
   
## 🧩 Структура выходного файла
Была определена заранее в ТЗ
```
{ 
    "code": 900,
    "from": "9000000", 
    "to": "9009999", 
    "operator": "МегаФон", 
    "region": "Московская область", 
    "city": null, 
    "district": null, 
    "timezone": "UTC+0300" 
}
```

## 📝 Примечания
- Если `region` — Москва или Санкт-Петербург, то значение переносится в поле `city`, а `region` становится `null`.
- `timezones.py` — содержит соответствие регион → временная зона
